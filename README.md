# CDA
This Program is a Simple Communication Debug Assistant (CDA) Project.   
CDA项目是Windows平台下简单的通信调试助手工程。

## 图标
![](https://github.com/Alopex6414/CDA/raw/master/Photo/Cat.png)

## 概述
CDA项目是Windows平台下简单的通信调试助手工程。CDA可以调试串口、网络通信数据。串口调试支持多种串口协议如RS-232、RS-422、RS-485，支持有线或无限的数据发送如串口、蓝牙、2.4GHz射频；实时曲线暂时只支持串口曲线，可以最多支持8条串口曲线的同时接收，可以导出数据到Excel或者截图保存。CDA可以在32bit和64bit环境下部署，支持Windos7/8/10。
